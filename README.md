## Spring Boot

Proyecto Spring Boot con las dependencias / starters:
* H2
* Spring Data JPA
* Spring Web
* Spring Boot Dev tools

API REST con acceso a BD H2 para persistencia de informacion.

Se puede realizar acceso a traves de Postman o Navegador.
